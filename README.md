# projetoEstoque

MVP - Sistema de Gestão de Estoque (Node.js + Express + Sequelize + React)

Setup rápido (backend com PostgreSQL):

1. Crie um banco PostgreSQL e um usuário com permissões.
2. Copie `.env.example` para `.env` e ajuste `PSQL_*` com suas credenciais.
3. Instale dependências:

```bash
npm install
```

4. Inicie o servidor backend:

```bash
npm run server
```

O frontend (React) pode ser rodado com `npm start` na raiz do projeto.

Observação: o projeto foi ajustado para usar apenas PostgreSQL conforme solicitado.
